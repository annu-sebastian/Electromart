# ElectroMart 🛒

ElectroMart is a modern, fully responsive e-commerce front-end application built for purchasing electronic gadgets and home appliances.

## 🌐 Live Demo
You can view the live project here: [https://annu-sebastian.github.io/Electromart/](https://annu-sebastian.github.io/Electromart/)

## ✨ Features
* *Interactive Shopping Cart:* Seamless product navigation and detailed cart views.
* *User Authentication:* Complete login and registration templates.
* *Seller Dashboards:* Dedicated layouts for sellers to register and manage products.
* *Responsive Design:* Optimized for mobile, tablet, and desktop screens.

## 🛠️ Technologies Used
* *HTML5:* Structuring the web pages and layouts.
* *CSS3:* Custom styling, responsiveness, and modern UI transitions.

## 📂 Project Structure
* index.html - Home page with featured categories.
* products.html & product-details.html - Product listing and specification views.
* cart.html & checkout.html - Cart overview and payment gateway wireframe.
* login.html & register.html - User onboarding modules.
* seller-dashboard.html & seller-register.html - Merchant portal interfaces.
